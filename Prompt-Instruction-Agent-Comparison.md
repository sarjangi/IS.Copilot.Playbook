# Prompt vs. Instruction vs. Agent vs. Agent Instruction

Ah! Let’s make a clear comparison of all four: prompt, instruction, agent, and agent instruction.

---

## 1. Prompt

**Definition:** The immediate input you give to an AI to get a response.

**Purpose:** Ask a question or request an action.

**Scope:** One-time; does not define behavior beyond that request.

**Example:**

"Explain photosynthesis in simple terms."

---

## 2. Instruction

**Definition:** A guideline or command within a prompt that tells the AI how to respond.

**Purpose:** Shape the format, style, or tone of the AI’s answer.

**Scope:** Usually applies only to that prompt unless combined with an agent instruction.

**Example:**

"Explain photosynthesis in simple terms using bullet points and short sentences."

Notice the bolded part is the instruction inside the prompt.

---

## 3. Agent

**Definition:** An AI system or entity that can act autonomously to perform tasks or interact with users.

**Purpose:** Acts as the “doer” that can respond to prompts and instructions.

**Scope:** Persistent; can handle multiple prompts over time.

**Example:**

A travel assistant AI that can answer questions, suggest itineraries, or book hotels.

---

## 4. Agent Instruction

**Definition:** A broader set of rules or guidelines defining the agent’s behavior, role, or personality.

**Purpose:** Ensure the agent responds consistently and appropriately across multiple prompts.

**Scope:** Persistent; affects every response the agent gives until changed.

**Example:**

**Agent Instruction:** "You are a friendly science tutor. Always explain topics in simple terms, give examples, and never use overly technical language."

**Prompt:** "Explain photosynthesis."

**Result:** The AI responds using the rules set by the agent instruction—polite, simple, and example-based.

---

## Quick Comparison Table

| Term | Definition | Purpose | Scope | Example |
|---|---|---|---|---|
| Prompt | What you ask AI | Get a response/action | One-time | "Explain photosynthesis." |
| Instruction | Guideline inside a prompt | Shape style, tone, format | One-time | "Use bullet points." |
| Agent | AI entity/system | Perform tasks | Persistent | Travel assistant AI |
| Agent Instruction | Rules for agent | Guide behavior/personality | Persistent | "Be polite, provide examples, no medical advice." |

## ?? Analogy

- **Agent** = the actor on stage (the AI).
- **Agent Instruction** = the actor’s role/personality (how they act consistently).
- **Prompt** = the scene the director calls (what to do this moment).
- **Instruction** = a note on how to perform the scene (style, tone, props).
